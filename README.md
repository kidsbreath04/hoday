# キッズブレス LP (Noe-LP)

放課後等デイサービス「キッズブレス」の紹介用ランディングページです。

## プロジェクト構成

- `index.html`: トップページ
- `pages/`: 下層ページ（放デイについて、会社概要、協力者への感謝など）
- `assets/`: 画像、CSS、JavaScriptなどの静的資産

## デプロイ方法

このプロジェクトは GitHub Pages で公開するように設定されています。

1. GitHub リポジトリにこのファイルをプッシュします。
2. GitHub のリポジトリ設定（Settings > Pages）から、`main` ブランチの `/root` を公開ソースとして選択します。
3. `.nojekyll` ファイルが含まれているため、静的ファイルが正しく配信されます。

## 注意事項

- お問い合わせフォームのリダイレクト先（`pages/contact.html` 内の `_next`）は、公開後のURLに合わせて調整してください。
